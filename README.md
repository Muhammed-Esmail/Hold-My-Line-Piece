# Hold-My-Line-Piece
Hold My Line Piece: Look Ma, no hands! (Wait, no... ONLY hands).

## Tetris Game controlled by handtracking
### How to play:
1. Clone the repo
2. Create a virtual environment
3. Install dependencies in the requirements.txt file
4. Run the Launcher.py python file and you're done!

## In Progress:
- Exporting game into just an executable for ease of use
- Better input handling

## A warning
- The game now simulates pressing arrow keystrokes so be careful.
